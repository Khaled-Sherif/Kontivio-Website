## Description
Kontivio is a web platform designed to provide remote customer support outsourcing services. This frontend application allows agents to interact with clients, manage tasks, and monitor service quality efficiently.

Key features include:
- User-friendly interface for agents and admins
- Real-time monitoring of customer interactions
- Dashboard to track service metrics
- Integration with backend APIs for task and client management

## Installation
Follow these steps to run the project locally:

1. **Clone the repository**
```bash
git clone https://github.com/Khaled-Sherif/Kontivio-Website.git
cd Kontivio-Website/my-app
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
